5. 현재 RLS 상태 (중요 — 다음 작업 0-6의 출발점)
현재 모든 테이블에 두 개 정책 존재
anon_full_access       | ALL | {anon}          | USING (true)
anon_full_access_auth  | ALL | {authenticated} | USING (true)
즉 모든 데이터가 로그인 여부와 무관하게 완전 공개 상태. 프로토타입 단계 임시 설정이며, 작업 0-6에서 전면 재설계 예정.
RLS 보안 부채

모든 테이블의 anon에 완전 접근 허용
로그인한 authenticated도 마찬가지로 완전 접근
employees.password는 'MANAGED_BY_AUTH' 플레이스홀더라 노출돼도 안전함
그러나 다른 테이블 데이터는 여전히 공개 상태
작업 0-6의 목표: 이 두 정책을 제거하고 role 기반 세밀한 정책으로 교체
