세션 15 시점에 supervisor가 role enum 값인지, 별 컬럼인지, is_team_leader와의 관계가 어떤지 미확인. 트랙 J 단계 1 (조사)에서 grep으로 자동 해소 예정.

해소 결과 따라 단위 1 마이그레이션 SQL + 단위 4 직원 표 "반장" 컬럼 렌더링 영향:
- supervisor가 role enum 값이면 → 기존 활성화만 + 신규 LAB장/HUB장 시드에 박음
- supervisor가 별 컬럼이면 → 컬럼 활용 정책 결정
- is_team_leader와 별 개념이면 → 분리 유지
- is_team_leader와 supervisor 관계에 따라 단위 4 표 "반장" 컬럼 표기·수정 로직 결정

## 7. 챗봇 운영 가이드 (세션 14 HANDOVER에서 이전, 임시 보관)

> ⚠️ 이 절은 임시. H-4 또는 H-5 마감 후 별도 `docs/operations/CHATBOT_USER_GUIDE.md`로 분리 권장. 현 시점에는 base 자료로 HANDOVER에 보관. 세션 14 HANDOVER (2026-04-14.md) 7절 그대로.

### 7.1 노출 조건

farm_admin / hr_admin / master 권한 사용자만 우하단 FAB(💬) 표시.

**세션 16 트랙 J-3 마감 후**: general 추가 (4종).

### 7.2 사용 방법

- FAB 클릭 → 챗봇 패널 열림
- 환영 메시지 + 예시 질문 3개 표시
- 예시 질문 클릭 → 즉시 전송
- 직접 입력 → Enter 전송 (Shift+Enter 줄바꿈)
- 한국어 입력 중 Enter는 글자 확정 (자동 전송 안 됨)
- 최대 500자

### 7.3 응답 종류

- 운영 데이터 조회 (출근, 휴가, TBM, 대기 승인, 사용자 목록)
- 피드백 제출 (버그 신고, 개선 의견)
- 일반 대화 (제한적)

### 7.4 에러 발생 시

"일시적인 오류입니다" 또는 다른 에러 메시지 표시 시:
- 잠시 후 다시 시도
- 지속 발생 시 관리자 문의 (master 권한)

**관리자(태우) 안내**: Anthropic API 결제 이슈 시 크레딧 충전 후 5~10분 지연 반영. 사용자에게는 "일시적인 오류" 메시지로 노출되니 충전 후 잠시 후 재시도 안내.
