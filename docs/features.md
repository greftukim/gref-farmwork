# 기능 상세

## 기능 구조 — 3개 층

### 1층: 인사 관리 (HR 기본)

**직원 정보 관리**
- 이름, 사번, 연락처, 직무(재배/관리/기타), 입사일
- 근로 조건 (주 40시간 기본)
- 직원 등록/수정/비활성화

**근무 일정**
- 주간 단위 근무 스케줄 등록
- 작업자별 가로 타임라인 뷰 (오전/오후 + 작업 내용 블록)
- 대부분 동일 시간대(08:00~17:00), 복잡한 시프트 관리 불필요

**출퇴근 기록**
- 앱에서 출근/퇴근 버튼 터치 (GPS 불필요, 현장 고정)
- 일별 근무 시간 자동 계산
- 지각/조퇴/결근 자동 판정
- 월별 근무 시간 집계

**휴가 관리**
- 연차 / 오전반차 / 오후반차 / 특별휴가
- 총 휴가 일수, 사용 일수, 잔여 일수 자동 계산
- 신청 → 관리자 승인/반려 워크플로우
- 팀 전체 휴가 현황 캘린더

### 2층: 농작업 관리 (현장 전용)

**작업 계획 및 배정**
- 관리자가 일별/주별 작업 계획 수립
- 작물별 작업유형 매핑
- 구역(zone) + 열(row) 단위 배정

**기본 작물 4종 및 작업유형**
- 토마토: 수확, 유인·결속, 적엽, 병해충 예찰, EC/pH 측정, 수분 작업
- 오이: 수확, 유인·결속, 적엽, 병해충 예찰, EC/pH 측정
- 미니파프리카: 수확, 유인·결속, 적엽, 병해충 예찰, EC/pH 측정, 수분 작업
- 딸기: 수확, 러너 정리, 적엽, 병해충 예찰, EC/pH 측정

**작업 기록**
- 작업 시작/종료 타이머 (소요 시간 자동 계산)
- 작업량 입력 (수량 + 단위: 매, 개, kg 등)
- 구역/열 범위 기록 (예: A동 1-8열)
- 사진 첨부, 메모

**생육 조사** (주 1~2회)
- 측정 위치: 구역, 열 번호, 주 번호
- 측정 항목: 초장(cm), 경경(mm), 엽수, 개화 화방 번호, 착과수, 과중(g)
- 이전 측정값과 비교 표시

**이상 신고**
- 사진 촬영/갤러리 첨부
- 구역 선택, 이상 유형 (병해충/작물이상/시설이상/기타)
- 코멘트, 관리자 확인 처리

**긴급 호출**
- 작업자 → 관리자 (긴급호출/질문확인/장비이상)
- 메모 첨부

**성과 분석**
- 작업자별 작업량/속도/완료율
- 작물별, 작업유형별 평균 소요 시간 비교
- 예상 vs 실제 소요 시간 평가

**공지사항**
- 관리자 작성 (일반/중요/긴급), 작업자 열람

## 화면 구성

### 관리자 (웹)

대시보드: 출근 현황, 작업 진행, 처리 필요 알림, AI 제안 알림
직원 관리: 직원 목록 테이블, 등록/수정
근무 관리: 주간 타임라인 뷰, 출퇴근 기록, 월별 집계
휴가 관리: 승인/반려, 잔여 현황 테이블, 팀 캘린더
작업 관리: 작업 계획(달력/리스트), 배정, 현황 보드, 작물 관리
기록 조회: 작업 이력, 생육 조사, 이상 신고, 수확 기록
통계 분석: 근무 시간, 작업 성과, 수확량 추이
공지사항: 작성/관리

### 작업자 (모바일 PWA)

홈: 출퇴근 버튼, 오늘 작업 목록, 긴급 호출 버튼
작업: 작업 카드, 시작/완료(타이머+수량), 주간 스케줄
생육 조사: 입력 폼, 이전값 비교
근태: 캘린더+휴가 신청, 이력, 근무 시간 확인
이상 신고: 작성(사진+유형+코멘트), 이력
공지사항: 열람
